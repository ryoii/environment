# Environment

Linux server environment backup.

# Usage

```
$ git clone https://github.com/ryoii/environment.git

$ cd environment

$ chmod +x install.sh

$ ./install.sh
```

# Update

```
$ cd environment

$ git pull

$ ./install.sh
```
